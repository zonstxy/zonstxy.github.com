安卓工程一键打包脚本

发布时间 : 2019-10-11
上一篇 : 平台打包SDK注意事项 下一篇 : 麻将游戏添加切牌动画

运行环境：Python 3 ，使用前请确保运行环境配置无误

# 准备工作

使用前，需要准备以下内容：

  • sdk.json 文件编辑无误
  • icon文件放于指定位置

# JSON文件编辑

整个打包过程的核心就是sdk.json文件，内容包括三方面：sdk配置、app配置、签名文件配置。

# sdk配置示例：

{
    "name": "wechat",	//固定不改，分别有wechat、baidu、bugly，xianliao、cajian
    "app_id": "wxe03ecdb6a7ca7486",		//申请的App_id或Key
    "secret": "9519f3aaa04cb9594fa055613f756bae"	//secret，没有写空即可
}

# app配置示例：

{
    "name": "package",							//固定不改
    "package_name": "com.baoqingmajiang.hall",	//包名
    "app_name": "中至麻将"						 //APP名称
}

# 签名配置示例：

签名文件放于frameworks\runtime-src\proj.android-studio目录下，打开cmd/powershell输入指令:

keytool -list -keystore 签名.jks -storepass 存储密码

即可查看对应的别名和数字指纹等信息。

 {
    "name": "signconfig",					//固定不改
    "keyAlias": "xq5",						//别名
    "keyPassword":"xq5",					//验证密码
    "storePassword":"zhongzhi",				//存储密码
    "signname":"xq5.jks"					//签名文件名
 }

配置好的文件放于frameworks\runtime-src\proj.android-studio\app路径即可，命名统一采用“sdk_示例.json”的格式。

# Icon图标放置

使用图标生成工具 Resize.exe 生成各尺寸的图片，在frameworks\runtime-src\proj.android-studio\app\icon路径下创建文件夹，取包名的第二段为文件夹名字，以com.xq5.zhongzhimj为例，创建xq5文件夹，放入各尺寸文件即可。

# 使用

目前支持单个平台的拆分切换，只需保证对应的签名文件、json配置文件、游戏icon以指定格式放于目录下，并将脚本文件置于frameworks\runtime-src\proj.android-studio\app路径运行即可

另外frameworks\runtime-src\proj.android-studio\app\assets下对应的文件请根据需要自行替换
